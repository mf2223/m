دليل سريع لتشغيل المتجر المحلي
--------------------------------
المحتوى داخل الحزمة:
- index.html
- styles.css
- app.js
- products.json
- README.txt

كيفية التشغيل محلياً:
1. فك الضغط عن الحزمة.
2. افتح index.html في متصفحك (أفضل تشغيل عبر خادم محلي لعمل fetch: 
   يمكنك تشغيل: python -m http.server 8000
   ثم افتح: http://localhost:8000)
3. لتعديل المنتجات، حرر products.json.
4. لربط بوابة دفع حقيقية (مثل PayPal أو شهادة الدفع المحلية)، ستحتاج إلى دمج API جهة الدفع في app.js أو استخدام منصة (Shopify/WooCommerce).

نصائح نشر:
- يمكنك رفع الملفات إلى GitHub Pages أو أي استضافة ملفات ثابتة.
- إذا أردت متجر كامل مع إدارة طلبات ودفع وبطاقات ائتمان، أنصح باستخدام WooCommerce (WordPress) أو Shopify.

إذا تبي، أقدر أجهز لك:
- نسخة متصلة بووكميرس أو شوبفاي.
- صفحة سياسة الخصوصية، الشحن والإرجاع بالعربية.
- شعار بسيط أو صور منتجات.
